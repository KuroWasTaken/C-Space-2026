# C-Space-2026
Le C’Space 2026 est un concours spatial étudiant organisé par Planète Sciences et le CNES. Du 11 au 17 juillet 2026, des équipes conçoivent et lancent fusées, CanSats ou expériences, vivant une immersion unique dans l’ingénierie spatiale.
